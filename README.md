# Task Manager API

[![Python](https://img.shields.io/badge/python-3.11+-blue.svg)](https://www.python.org/)
[![FastAPI](https://img.shields.io/badge/FastAPI-0.110+-009688.svg)](https://fastapi.tiangolo.com/)
[![CI](https://github.com/sonnymay/TasksManagerAPI/actions/workflows/ci.yml/badge.svg)](https://github.com/sonnymay/TasksManagerAPI/actions/workflows/ci.yml)

Clean REST API for task management, built with FastAPI, SQLAlchemy, and SQLite/PostgreSQL-compatible models.

## What this code shows

- CRUD API with request/response validation through Pydantic.
- SQLAlchemy models and session dependency wiring.
- Swagger/OpenAPI docs generated by FastAPI at `/docs`.
- CORS middleware for browser frontend integration.
- Tests for the main task CRUD flow against an isolated SQLite database.
- Small deployable Python API shape: `app/main.py`, `app/routes/tasks.py`, `app/models.py`, `app/schemas.py`.

## Endpoints

| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/tasks/` | List tasks |
| `POST` | `/tasks/` | Create task |
| `GET` | `/tasks/{task_id}` | Get task |
| `PUT` | `/tasks/{task_id}` | Update task |
| `PATCH` | `/tasks/{task_id}/complete` | Mark complete |
| `DELETE` | `/tasks/{task_id}` | Delete task |

## Local development

```bash
git clone https://github.com/sonnymay/TasksManagerAPI.git
cd TasksManagerAPI
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Open `http://localhost:8000/docs` for interactive API docs.

## Quality check

```bash
pip install -r requirements-dev.txt
pytest
python -m compileall app
```
